# Sex-biased herbivory
This repository contains data and code for the analyses presented in the paper "Can feeding preferences of sequential herbivores reinforce female-biased sex ratios in an alpine willow?" currently under review.

Our study investigates the foraging preferences of two common alpine herbivores—a caterpillar and a small mammal—feeding on the widespread dioecious tundra shrub Salix arctica at different times during the growing season. Using cafeteria-type experiments, we test whether herbivores exhibit sex-biased feeding and show that caterpillars preferentially consume leaves from male plants in early summer, whereas collared pikas show no preference between male and female leaves later in the season.
